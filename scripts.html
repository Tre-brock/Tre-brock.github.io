Install Notepad++ on Linux

Notpad++ is a linux only app. With this script it downloads the exe, runs it in wine, opens the installer and then deletes the exe and sends you a notifaction when it's done installing.

Type "bash <(curl -s https://raw.githubusercontent.com/Tre-brock/Notepad-plus-plus-linux/master/notepad%2B%2B%20linux%20installer.sh)" No quotes
