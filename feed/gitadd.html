I made a new browser addon to view github notifactions(Mon Apr 29 2024)

I made a new addon that lets you view GitHub notifications and messages from clicking on the addon. Now you don't have to have a tab for checking on your messages. I don't use GitHub that much anymore so it is nice not having to have it open for a notification here and there. The reason reason this only supports Pale Moon and Seamonkey is they support legacy addons format.
